I have written in Java or JavaScript, translated from Java into JavaScript, or in a few instances adapted from JavaScript more than 200 algorithms. The algorithms are collected in a repository on my GitHub account. In another repository, there are my implementations of introductory programs of neural networks. Other projects on GitHub include an adaptation of a web crawler in JavaScript and a series of grid games that are templates built from a template. A description of each and its link can be found on my resume. Of particular note among them is a grid game that uses Dijkstra's algorithm to find the shortest path in a grid of elements between an opponent and player as they move on a wraparound board. In August 2020, I wrote a book titled Algorithms and Programs.

I wrote a theoretical treatment of cancer that I believe is viable and in need of testing. The paper can be found at this website.

I write on Twitter about the scientific topics of climate change, evolution, and expansion.
